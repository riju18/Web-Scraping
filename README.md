## Learning Project

## Setup Environment:
- create a virtual environment;
- Install requirements.txt.;
- place the above two dirs into newly created venv;

## Run the project:
- Goto specific dir;
- cmd: scrapy crawl spider_name or scrapy crawl spider_name -o *.ext [ext: json, csv, xml]
